use the script in "\DevOps Configuration Mgmt\ARM\self signed cert\servicefabric cluster prerequisites.ps1" 
to generate 2 self signed certificates and import them in current user's personal store:
- fabricclusterlinux-pri.pfx
- fabricclusterlinux-sec.pfx

fabricclusterlinux is the DNS name of the cluster, so it must be unique to the Azure Region you're deploying into later.
Change it accordingly.
